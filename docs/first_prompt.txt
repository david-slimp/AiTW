This new project will be a multiplayer online text adventure game (highly modular and extendible server written mostly in python with html5/TS/CSS web interface for the players)
A brief description is:  Christian-based MMO / MUD (text-based like Zork with nice web interface). Some of the main design goals will be that we try to keep the code extremely highly modular so that all rooms, objects, commands, etc are defined and set in modular files and not hardcoded. Adding a new room or item should be as simple as placing a new room or item file in a directory and the server will see it, and start using it. Admins can use online commands to create new rooms, items, etc.
The Christian-based aspect and gameplay will be focused on the player starting off to be a new Christian, and then plays and takes actions to grow in faith, understanding, wisdom, conviction, etc.... ultimately taking actions that lead to producing Fruit of the Spirit, acts of service, evangelism, etc.  One of the underlying goals should be to help real human players learn and better understand what they can realistically do in the real world to better serve God and take a more active and accurate role in living a lifestyle according to The Way of Jesus.
